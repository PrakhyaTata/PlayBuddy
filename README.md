PlayBuddy IOS application[Swift 3,Xcode,FireBase,GoogpleMaps API]
– PlayBuddy is a IOS application that connects people around with same interest of sports.A user can call or send message
to others on his list fromthe application.
– GoogleMaps API is used to find the playgrounds around you and details its location inMapView.
– Firebase is used to store the user information and images they upload.
